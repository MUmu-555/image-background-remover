## 1. 项目背景与目标

### 1.1 背景

"Image Background Remover" 是一个高搜索量的工具型需求词。用户需要一个免费、快速、无需注册的在线抠图工具，用于电商主图、证件照、设计素材等场景。

### 1.2 MVP 目标

- 验证工具型网站的用户留存与转化能力
- 以最小成本上线，获取自然流量（SEO）
- 核心路径：**上传图片 → AI 去背景 → 下载 PNG**，全程无需注册

### 1.3 成功指标

| 指标 | MVP 目标 |
|------|---------|
| 页面加载时间 | < 2s（Cloudflare CDN 全球加速） |
| 抠图响应时间 | < 5s |
| 用户任务完成率 | > 80%（上传到下载） |
| 月活（上线后第1个月） | > 500 UV |

## 3. 功能范围

### 3.1 MVP 包含（Must Have）

#### F1 — 图片上传
- 支持点击上传（文件选择器）
- 支持拖拽上传（Drag & Drop）
- 支持格式：JPG、PNG、WEBP
- 文件大小限制：≤ 20MB
- 不支持多文件同时上传（MVP 阶段单张处理）

#### F2 — AI 背景去除
- 调用 Remove.bg API 处理图片
- 图片在内存中处理，不落盘存储
- 显示处理进度状态（Loading 动画）
- 处理失败时展示友好错误提示

#### F3 — 结果预览
- 处理完成后展示"原图 vs 去背图"左右对比
- 去背图使用棋盘格背景展示透明区域

#### F4 — 下载
- 下载格式：PNG（带透明通道）
- 文件名：`removed-background.png`
- 一键下载，无需登录

#### F5 — 再次处理
- 提供"Remove Another"按钮，重置页面状态

#### F6 — SEO 基础
- 页面 Title、Description、Keywords 配置
- 语义化 HTML 结构
- 移动端响应式布局

---

## 4. 页面与交互设计

### 4.1 页面结构

```
┌─────────────────────────────────┐
│  Header：Logo + 产品名           │
├─────────────────────────────────┤
│  Hero：标题 + 副标题             │
├─────────────────────────────────┤
│                                 │
│  主区域（状态机驱动）            │
│  ┌─────────────────────────┐   │
│  │  idle: 上传区域          │   │
│  │  processing: Loading    │   │
│  │  done: 对比图 + 下载     │   │
│  │  error: 错误提示         │   │
│  └─────────────────────────┘   │
│                                 │
├─────────────────────────────────┤
│  特性说明：3 个卖点              │
├─────────────────────────────────┤
│  Footer                         │
└─────────────────────────────────┘
```

### 4.2 状态机

```
idle ──[上传文件]──▶ processing ──[成功]──▶ done
                          │                  │
                        [失败]           [再次上传]
                          ▼                  │
                        error ◀─────────────┘
                          │
                      [Try Again]
                          │
                          ▼
                        idle
```

### 4.3 错误处理

| 错误类型 | 提示文案 |
|---------|---------|
| 文件格式不支持 | "Please upload an image file (JPG, PNG, WEBP)." |
| 文件超过 20MB | "Image must be under 20MB." |
| API Key 未配置 | "Service not configured. Please try again later." |
| Remove.bg 余额不足 | "API credits exhausted." |
| 网络超时/请求失败 | "Failed to reach the processing service. Please try again." |

## 5. 技术架构

### 5.1 技术选型

| 层级 | 技术 | 原因 |
|------|------|------|
| 前端框架 | Next.js 14（App Router） | 全栈一体，SEO 友好 |
| 样式 | Tailwind CSS | 快速开发，无额外 CSS |
| API 运行时 | Edge Runtime | 兼容 Cloudflare，低延迟 |
| 第三方 API | Remove.bg | 效果最佳，接入简单 |
| 部署 | Cloudflare Pages | 全球 CDN，免费套餐够用 |
| 存储 | 无 | 图片不落地，全内存处理 |

### 5.2 API 数据流

```
用户浏览器
    │ POST /api/remove-bg（FormData: image）
    ▼
Next.js Edge API Route
    │ POST https://api.remove.bg/v1.0/removebg
    │ Header: X-Api-Key: ${REMOVEBG_API_KEY}
    ▼
Remove.bg API
    │ 返回 PNG 二进制流
    ▼
Edge Route 直接转发 ArrayBuffer
    │ Content-Type: image/png
    ▼
用户浏览器（Blob URL → 预览 & 下载）
```

### 5.3 环境变量

| 变量名 | 说明 | 必填 |
|--------|------|------|
| `REMOVEBG_API_KEY` | Remove.bg API 密钥 | ✅ |

## 8. 风险与应对

| 风险 | 概率 | 影响 | 应对方案 |
|------|------|------|---------|
| Remove.bg 免费额度用完 | 高 | 服务中断 | 监控用量，提前购买 Credits；或接入 Clipdrop 作为备用 |
| Cloudflare Workers 超时限制 | 中 | 大图处理失败 | 限制上传大小（≤20MB），提示用户压缩图片 |
| Remove.bg API 故障 | 低 | 服务不可用 | 展示友好错误页，说明服务维护中 |
| 竞品抄袭 | 高 | 流量分流 | 优化 SEO、提升速度和用户体验作为护城河 |
